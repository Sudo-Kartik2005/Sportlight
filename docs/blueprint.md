# **App Name**: Sportlight

## Core Features:

- Sport Selection: Allow users to select a sport of interest from a predefined list or type it in.
- Basic Sport Information: Display key information about the selected sport including history, rules, and key terms.
- Embedded Video Tutorials: Embed relevant YouTube video tutorials for the selected sport.
- AI Chat Coach: Provide a chatbot "tool" powered by AI to answer sport-specific questions.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), reminiscent of athletic wear and clear skies, conveying trust and dynamism.
- Background color: Light gray (#F0F0F0), offering a clean and modern backdrop to make content pop.
- Accent color: Energetic orange (#F26419), used to highlight key interactive elements such as buttons and links, drawing user attention.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to ensure a clean and modern reading experience.
- Use clear and universally recognizable icons related to sports and informational categories to enhance navigation and visual appeal.
- Implement a card-based layout for different information sections to organize content neatly and provide a user-friendly experience.
- Incorporate subtle transition effects when switching between sports or updating data, to provide a smooth and engaging user experience.
